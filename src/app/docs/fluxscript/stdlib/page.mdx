# Standard Library

FluxScript ships with a comprehensive standard library organized into modules. Each module is imported with `use` and provides domain-specific functionality for mathematics, signal processing, statistics, string manipulation, and complex matrix operations.

---

## `math.flux`

Core mathematical functions operating on scalars and vectors.

| Function | Description | Signature |
| :--- | :--- | :--- |
| `max(a, b)` | Returns the larger of two values. | `(double, double) -> double` |
| `min(a, b)` | Returns the smaller of two values. | `(double, double) -> double` |
| `clamp(x, lo, hi)` | Clamps `x` to the range `[lo, hi]`. | `(double, double, double) -> double` |
| `lerp(a, b, t)` | Linear interpolation between `a` and `b` by factor `t`. | `(double, double, double) -> double` |
| `rad2deg(r)` | Converts radians to degrees. | `(double) -> double` |
| `deg2rad(d)` | Converts degrees to radians. | `(double) -> double` |
| `sign(x)` | Returns the sign of `x` (`-1.0`, `0.0`, or `1.0`). | `(double) -> double` |
| `is_close(a, b)` | Checks if `a` and `b` are within `1e-9`. | `(double, double) -> bool` |
| `step(x)` | Unit step function (`0.0` if `x < 0.0`, else `1.0`). | `(double) -> double` |
| `factorial(n)` | Computes the factorial of `n` recursively. | `(double) -> double` |
| `gcd(a, b)` | Greatest common divisor using the Euclidean algorithm. | `(double, double) -> double` |
| `lcm(a, b)` | Least common multiple. | `(double, double) -> double` |
| `map_range(x, in_lo, in_hi, out_lo, out_hi)` | Remaps `x` from `[in_lo, in_hi]` to `[out_lo, out_hi]`. | `(double, double, double, double, double) -> double` |
| `smoothstep(x)` | Cubic Hermite interpolation clamped to `[0.0, 1.0]`. | `(double) -> double` |
| `round_to(x, n)` | Rounds `x` to `n` decimal places. | `(double, double) -> double` |
| `frac(x)` | Fractional part of `x` (`x - floor(x)`). | `(double) -> double` |
| `wrap(x, lo, hi)` | Wraps `x` into the `[lo, hi)` range. | `(double, double, double) -> double` |
| `pingpong(x, lo, hi)` | Bounces `x` back and forth between `lo` and `hi`. | `(double, double, double) -> double` |
| `inverse_lerp(x, a, b)` | Locates where `x` lies between `a` and `b`. | `(double, double, double) -> double` |
| `sigmoid(x)` | Logistic sigmoid function: `1.0 / (1.0 + exp(-x))`. | `(double) -> double` |
| `normalize_angle(x)` | Wraps angle `x` to the `[0, 2*pi)` range. | `(double) -> double` |
| `normalize_angle_sym(x)` | Wraps angle `x` to the `[-pi, pi]` range. | `(double) -> double` |
| `softplus(x)` | Smooth ReLU: `log(1.0 + exp(-abs(x))) + max(x, 0.0)`. | `(double) -> double` |
| `clamp01(x)` | Clamps `x` to `[0.0, 1.0]`. | `(double) -> double` |
| `relu(x)` | Rectified linear unit: `max(0.0, x)`. | `(double) -> double` |
| `logit(x)` | Log-odds function: `log(x / (1.0 - x))`. | `(double) -> double` |
| `avg(a, b)` | Arithmetic mean of two values. | `(double, double) -> double` |
| `comb(n, k)` | Binomial coefficient `n choose k`. | `(double, double) -> double` |
| `perm(n, k)` | Permutations `P(n, k) = n! / (n-k)!`. | `(double, double) -> double` |
| `reflect(v, n)` | Reflects vector `v` about unit normal `n`. | `(double, double) -> double` |

## Built-in Functions

FluxScript provides compiler-injected built-in math and matrix functions that do not require any imports.

### Math Intrinsics
Double-precision intrinsic operations available globally:

- Trigonometric: `sin(x)`, `cos(x)`, `tan(x)`, `asin(x)`, `acos(x)`, `atan(x)`, `atan2(y, x)`
- Hyperbolic: `sinh(x)`, `cosh(x)`, `tanh(x)`
- Exponential/Log: `sqrt(x)`, `exp(x)`, `log(x)`, `log10(x)`, `pow(base, exp)`
- Rounding/Numeric: `abs(x)`, `floor(x)`, `ceil(x)`, `round(x)`
- Constants: `pi()`, `e()`

### Matrix Operations
High-performance matrix/vector functions:

- `det(m)`: Determinant of a real matrix.
- `inv(m)`: Inverse of a real square matrix.
- `eig(m)`: Eigenvalues/eigenvectors decomposition.
- `dot(a, b)`: Dot product of two vectors.
- `cross(a, b)`: Cross product of two 3D vectors.
- `norm(v)`: Frobenius/Euclidean norm of matrix or vector.

Matrix creation functions:
- `matrix_zeros(rows, cols)`: Creates a zero-initialized matrix of double-precision elements. (Note: `zeros` is a reserved SPICE keyword, so this built-in is used instead).
- `matrix_ones(rows, cols)`: Creates a matrix initialized to ones.
- `matrix_eye(n)`: Creates an $n \times n$ identity matrix.
- `matrix_copy(m)`: Creates a deep copy of matrix `m`.

### Member Functions and Operators
- **Transpose Operator (`'`)**: Appending a single quote to a matrix transposes it (e.g. `A'`).
- **Element-wise Operators (`.+`, `.-`, `.*`, `./`, `.^`)**: Perform mathematical operations element-by-element across matrices of matching shapes.

### Neural Network Built-ins
Built-in AI/ML primitives (exposed directly from the runtime):
- `flux_nn_create(layers: matrix, learning_rate: double) -> double`: Creates a neural network with the specified layers configuration.
- `flux_nn_train(model: double, inputs: matrix, outputs: matrix, epochs: double) -> void`: Trains the network.
- `flux_nn_predict(model: double, inputs: matrix) -> matrix`: Evaluates predictions on the network.

## Importing Modules

Use the `use` keyword to import standard modules:

```fluxscript
use math
use trig
use array
use stats
use signal
use string
use cmatrix
```
