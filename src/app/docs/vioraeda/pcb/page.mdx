# PCB Editor

> **Status:** The PCB editor is currently in **demo/prototype stage**. Core functionality — trace routing, DRC, Gerber export, and 3D preview — is implemented. Some advanced features remain under active development.

VioraEDA's PCB editor provides a full board design environment built on the same rendering pipeline as the schematic editor. It supports copper pour, differential pairs, length tuning, a DRC engine, and Gerber export for fabrication.

## Board Items (8)

| Item | Description |
|---|---|
| Component | Placed component with associated footprint |
| Trace | Copper trace routing between pads |
| Via | Through-hole via connecting copper layers |

| Copper Pour | Filled copper zone on a layer |
| Shape | Geometric board outline or silkscreen shapes |

| Ratsnest | Airwire indicators for unconnected nets |

## Design Rule Check (DRC)

The DRC engine validates the board against a configurable rule set before fabrication. Rules are defined in the Board Setup dialog and include:

- **Clearance rules** — Minimum spacing between copper objects (trace-to-trace, pad-to-pour, via-to-trace)
- **Trace width rules** — Per-net minimum and maximum trace widths
- **Via size rules** — Minimum annular ring and drill diameter
- **Copper pour rules** — Pour-to-pad clearance and minimum spoke width
- **Custom rule editor** — Define constraint expressions for specific net classes

## Gerber Viewer

The integrated Gerber viewer provides layer-by-layer inspection of the generated fabrication files before submission:

- 2D layer visualization with per-layer color control
- 3D composite board preview
- Layer visibility and ordering controls
- Point-to-point measurement tools

## Import / Export

- KiCad `.kicad_pcb` footprint import
- Netlist import from the VioraEDA schematic editor
- Native `.flxpcb` file I/O

## Plugin System

The PCB editor supports native C++ plugins through the `pcb_plugin.h` interface. Plugins are loaded as shared libraries (`.so` on Linux, `.dll` on Windows) via `QPluginLoader` and can add custom tools, analysis engines, and export formats.
